ADDITION 1 — braveSearch() function (lines 398-455)
  Inserted above maybeRunWebLookup().

  async function braveSearch(query): Promise<WebSearchResult[]>
  - Reads BRAVE_API_KEY from process.env
  - Absent/empty key: returns [] silently, never throws
  - biasQueryForUK() applied to query before sending
  - GET https://api.search.brave.com/res/v1/web/search
    ?q=<query>&count=5&country=GB&search_lang=en
  - Header: X-Subscription-Token: <key>
  - 5s timeout (reuses TAVILY_TIMEOUT_MS)
  - Maps Brave result shape to WebSearchResult:
      obj["description"] -> content   (Brave uses "description" not "content")
      obj["title"]       -> title
      obj["url"]         -> url
  - Same MAX_SNIPPET_LEN cap as Tavily results
  - Returns [] on any error (network, timeout, bad JSON, non-200) — never throws
