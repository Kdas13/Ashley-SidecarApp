THE FULL PICTURE
================================================================================

Kane discovered that the "Her Profile" menu in Ashley Sidecar is a
universal system-level override. Anything stored there overrides the
model's defaults on every message. This makes it the correct control
plane for image generation governance in v1.

The goal is an 11-layer image builder inside the profile screen.
Each layer = one parameter of the image prompt. By the time the user
has tapped through all 11 layers, a complete unambiguous master prompt
has been built without typing a single word. OpenAI gets fully specified
instructions and has zero decisions to make on its own.

This eliminates portrait bias, wrong environments, wrong animals, and
wrong framing — because every parameter is explicitly locked before
the request reaches OpenAI.

Appearance drift is accepted. At wide angle / environment-centric
framing, Ashley occupies 5-15% of the frame. Blue hair and choker
carry identity at that scale. Facial consistency doesn't matter.

No R2 storage needed. No reference images needed for v1.
The funnel + correct routing is the complete solution.

AFTER FIXES CONFIRMED — UI REBUILD
================================================================================

Full spec is in: WREN_IMAGE_BUILDER_UI_REBUILD.txt

Short version: replace the current flat button rows with an 11-layer
linear funnel. Each layer unlocks the next. Auto on any layer = server
decides. The layers in order:

1. Composition Mode (Ashley-Centric / Balanced / Environment-Centric /
   Architectural / Social / Documentary)
2. Environment (Home / Food & Drink / Entertainment / Public / Outdoor
   / Travel / Creative / Seasonal — with sub-categories)
3. Atmosphere (Cozy / Romantic / Busy / Festive / Rustic / Lively etc)
4. Time of Day (Morning / Afternoon / Evening / Night)
5. Season & Weather (Spring-Winter / Clear-Snow)
6. Activity (Photography / Sports / Home / Food & Drink / Culture /
   Social / Cats)
7. Occupancy — Household (Ashley only / + Kane / + Cats / + Both)
   + Environmental (None / Sparse / Light / Moderate / Busy / Packed)
8. Relationships (Relaxed / Playful / Affectionate / Group Banter)
9. Camera & Framing (Mode / Lens / Awareness / Shot Distance /
   Orientation)
10. Occasion (Casual / Date Night / Birthday / Weekend Outing etc)
11. Confirm & Save — full summary of all selections, Save + Reset

Breadcrumb trail at bottom of every layer showing selections so far.
Tapping breadcrumb jumps back to that layer.
Back button on every layer.
Mobile-first tap targets — S24 Ultra is the primary device.
