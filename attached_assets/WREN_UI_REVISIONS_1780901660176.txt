REVISION 3 — GENERATE BUTTON IN THE UI
================================================================================

Add a "Generate test image" button at the bottom of the Image defaults
section, below the Reset button.

Behaviour:
- Kane taps "Generate test image"
- The current profile selections are used as the prompt parameters
- A single image is generated (not four — this is a preview)
- A loading spinner shows while generating
- The generated image appears inline in the profile screen directly
  below the button
- The image is tappable to enlarge (same as chat images)
- A "Send to chat" button appears below the generated image
  — tapping it sends the image into the main chat as if Ashley sent it
- The generated image does not auto-send to chat unless Kane taps
  "Send to chat"

This allows Kane to preview exactly what his selections produce
without going back to chat and typing a request.

The generate button uses the same image generation endpoint already
in use. No new backend endpoint needed — just call it with the current
profile governance params directly.
