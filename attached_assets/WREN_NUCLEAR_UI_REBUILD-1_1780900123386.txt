================================================================================
UI BEHAVIOUR
================================================================================

Each section is a collapsible accordion.
Tap the section header to expand or collapse it.
Expanded sections show all options as tappable rows with a checkbox.
Tapping a row toggles the checkbox and auto-saves immediately.
Single-select sections deselect the previous selection automatically.
Multi-select sections (Camera Mode only) allow multiple ticked options.

At the top of the Image defaults section show a summary line:
"Environment-Centric · Living Room · Evening · Wide Room"
showing the current active selections for quick reference without
expanding all sections.

A "Reset all to defaults" button at the bottom of the section.
Tapping it resets all sections to the defaults listed above and saves.

================================================================================
CRITICAL — HOW TO THINK ABOUT THIS
================================================================================

This UI is a manual image prompt builder.

Treat it exactly like typing a prompt into ChatGPT image generation.
There are no reference images. There is no assumed knowledge of who
Ashley is. The system has no memory of previous images. Every generation
starts from zero.

What Kane ticks in this UI becomes the prompt. Every ticked option is
one more line of instruction sent to OpenAI. The more specific the
selections, the more specific the output.

If Kane ticks:
  Environment-Centric + Living Room + Evening + Wide Room + Unaware +
  Ashley only + Relaxing

The prompt that goes to OpenAI should read something like:
  "Wide angle photograph of a lived-in living room in the evening.
  The room is the subject. A young woman with blue wavy hair and a
  black lace choker is present in the scene, occupying roughly 10-15%
  of the frame. She is relaxing, unaware of the camera. Documentary
  photography style. Natural lighting. Film grain."

That is a ChatGPT image prompt. Built entirely from Kane's checkbox
selections. No assumed knowledge. No reference images. No shortcuts.

Every section Kane ticks adds specificity to that prompt.
Every section left on Auto gets a sensible default inserted by the server.
Portrait and Selfie unticked means those words never appear in the prompt.

Build the UI as a prompt construction tool, not a settings panel.
